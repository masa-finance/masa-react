## Setup

```
git clone https://github.com/masa-finance/masa-react.git

sudo nano /etc/hosts
// add 127.0.0.1 masatools.finance

brew install mkcert

cd masa-react
mkcert masatools.finance

yarn 
yarn storybook
```
